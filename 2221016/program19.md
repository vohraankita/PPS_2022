## Program 25 : Write a program using string reverse functions.
```C
#include<stdio.h>
#include<string.h>
int main()
{
	char a[25]="MADAM";
	printf("The given string is = %s",a);
	printf("\nAfter reversing string is = %s",strrev(a));
	return 0;
}
```
**Output :**
```C
The given string is = MADAM
After reversing string is = MADAM
